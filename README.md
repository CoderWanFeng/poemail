## 📚简介

``poemail``是用Python自动发邮件的工具箱

-------------------------------------------------------------------------------

## 📦安装

### 🍊pip 自动下载&更新

```
pip install -i https://mirrors.aliyun.com/pypi/simple/ poemail -U
```
